El proyecto se concluyó de manera exitosa con los puntos debidos y cumplidos de manera adecuada.
Se estructuró un primer programa que satisfaciera con un ciclo for y cambios mínimos para cada planeta, esto permitió una misma estructura para cada archivo C que leyera distintas filas del "texto1" de acuerdo a los parámetros y/o variables que comprendía cada planeta. El valor de 'h' determinó por ciclos de Euler, la descripción de la órbita de cada planeta, cuando este tenía un tiempo de trayectoria de 1 día, 101 días, 201 días y 365 días, cada resultado se aprecia en la pantalla usuario de cada programa.
Se elaboró un diagrama de flujo que describe detalladamente las estructura del programa base, con códigos en el diagrama para explicar de la manera más explícitamente posible esta estructura.
Finalmente, con los datos de las posiciones en cada valor de h, se elaboró una secuencia de puntos cuando el planeta tenía trayectoria en 1 día, 101 días, 201 días y 365 días, luego para cada planeta con estos puntos se obtuvo de manera analítica la ecuación general de la elipse que describe la órbita de cada uno, representada en el archivo de gráficas.
